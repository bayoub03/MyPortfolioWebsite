# bouhnine.com

Personal portfolio website for Ayoub Bouhnine — Cyber Security Incident Response Analyst at Proximus Ada, M.Sc. Cybersecurity from ULB Brussels.

## What's in here

A single-page static site built with vanilla HTML, CSS, and JavaScript (no frameworks, no build step). Covers experience, education, projects, CTF write-ups, certifications, testimonials, and contact info.

## Structure

```
index.html   – the entire page
css/style.css – styling (light/dark theme via CSS custom properties)
js/main.js   – interactions (theme toggle, nav, scroll animations)
```

## Running locally

Open `index.html` in a browser. No server required.
